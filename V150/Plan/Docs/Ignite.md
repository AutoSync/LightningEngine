# Random Ignite

O Random Ignite é o editor visual de scripts da Lightning Engine, projetado para permitir que os desenvolvedores criem e editem scripts de forma visual e intuitiva, sem a necessidade de escrever código manualmente. Ele se integra ao Nucleo Script Engine, permitindo que os scripts criados no Random Ignite sejam executados diretamente na engine, oferecendo uma experiência de desenvolvimento fluida e eficiente. O Random Ignite deve ser projetado para ser fácil de usar, permitindo que os desenvolvedores criem scripts complexos e personalizados sem precisar de conhecimento profundo sobre programação ou a estrutura interna da engine.

## Parser

O O Script Visual Random Ignite se conecta ao Nucleo Script Engine, permitindo que os scripts criados no editor sejam interpretados e executados diretamente na engine. Ele deve ser projetado para ser fácil de usar, permitindo que os desenvolvedores criem scripts complexos e personalizados sem precisar de conhecimento profundo sobre programação ou a estrutura interna da engine.

C# e Random Ignite são as principais linguagens de script suportadas, mas o editor deve ser projetado para ser flexível o suficiente para suportar outras linguagens de script no futuro, como Lua ou Python. Nenhuma linguagem roda codigo nativo, tudo é interpretado pela VM do Nucleo, garantindo segurança e portabilidade dos scripts criados na Lightning Engine.
