# Engine Core

Baseado no roadmap v0.15.0, com status unificado por contexto.

## Concluído

- Window (SDL_Window + SDL_GPUDevice + loop com delta time)
- GameInstance (ciclo completo)
- Renderer base (queue-based)
- Engine Settings (`Types.h`)
- ClearColor configurável
- InputManager completo
- Pipeline 2D funcional
- Texture2D (load/draw/tint)
- Integração do pipeline 3D no renderer
- Framebuffer base

## Pendente

- Evoluções de render pipeline avançado em módulos específicos
