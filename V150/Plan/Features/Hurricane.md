# Hurricane — Sistema de Partículas

Baseado no roadmap v0.15.0, com status unificado por contexto.

## Concluído

- Particles — Hurricane (`Hurricane.h`)
- Pool fixo de partículas
- `emitRate`, `lifespan`, `gravity`
- Interpolação de tamanho e cor (`lerp`)
- Render com `FillCircle`
- `ParticleSystemComponent`

## Pendente

- Integração expandida com Equinox (materiais avançados)
- Integração planejada com MotionPhysix
- Extensões avançadas via editor/script
