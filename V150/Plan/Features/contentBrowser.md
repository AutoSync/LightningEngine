# Content Browser

Baseado no roadmap v0.15.0, com status unificado por contexto.

## Concluído

- Leitura de assets do disco via `fs::recursive_directory_iterator`
- Integração com drag & drop para cena/viewport
- Convenção de iconografia vinculada ao browser (`cb_*`)
  - `cb_add`, `cb_undo`, `cb_redo`, `cb_saveall`, `cb_folder`, `cb_openfolder`

## Pendente

- Normalização de assets para formato `.leasset`
- Melhoria de pipeline de import/export por tipo de asset
- Exibir thumbnails dedicadas por tipo de asset no grid/lista

## Convenção

- Prefixo `cb_` é exclusivo do Content Browser.
- Referência oficial: `Plan/Features/IconografiaEditor.md`.
